### 3. 运输与安装页（Shipping & Installation）

**补充以下关键信息：​**

#### 叉车吨位要求

- 75 kW 及以下 → 3 吨叉车
- 90 kW ～ 132 kW → 5 吨叉车
- 160 kW ～ 250 kW → 10 吨叉车
- 亦可使用行车起吊，需在叉车孔内插入适合槽钢，并使用吊索撑开器

#### 运输注意事项

- 运输过程中不可碰撞，以免损坏设备壳体
- 运输过程中不可堆码，可能会造成壳体变形
- 运输过程中需做好防潮措施，严重潮湿可能会影响设备性能

#### 安装场地要求

- 室内需保证良好通风，否则可能导致高温停机
- 进气温度或环境温度不得低于 0°C，低于此温度须联系制造商
- 机器应放置于空气凉爽、清洁的环境中，如有必要安装吸气管道
- 地面不平坦或倾斜时，须向制造商咨询

#### 配管尺寸说明

- 气体在管内流动会产生摩擦阻力与局部阻力，导致压力损失
- 实际压力降公式：**实际压力降 = 表值 × 管长 ÷ (100 × 压缩比)​**，其中压缩比 = 表压力 + 1
- 弯头、异径接管、三通接头、阀门等产生的局部压力损失需另行查阅相关手册叠加计算

#### 推荐系统配置（建议配图）
空压机 → 储气罐 → 冷干机 → C级过滤器 → T级过滤器 → A级过滤器 → 用气端

### 4. 安全规范页（Safety Guidelines）

> ⚠️ 这是说明书中篇幅最大的章节，网站上必须完整呈现，建议分为三个子模块。

#### 电气安全

- 根据功率正确选择电源线径，线径过小会导致高温烧毁
- 建议空压机单独使用一套电力系统，避免与其他负载并联
- 电动机运转电流不得超过额定电流的 15%
- 三相电流不平衡时，最高与最低相电流差不得超过 5%
- 电压降不得低于额定电压的 5%
- 必须安装接地线，且不可接在空气输送管或冷却水管上
- 根据功率配置适当空气开关（断路器）

#### 操作安全

- 禁止将压缩空气对准人体皮肤或用于衣物除尘
- 声压级达到或超过 90 dB(A) 的环境须佩戴护耳器
- 所有箱体门在运行过程中必须保持关闭
- 安装远程控制时，机器必须标注警告："此机器由远程控制，可能在无警告的情况下起动"
- 进气口不得吸入易燃气体、蒸汽或颗粒（如涂漆溶剂）
- 不得低于或高于额定限值运行机器
- 使用压缩空气清洁设备时，必须佩戴防护眼镜

#### 维修前必须执行的停机程序（严格按顺序操作）

1. 停止运行压缩机
2. 关闭空气出口阀门
3. 按下紧急停机按钮
4. 切断电源，在电源开关处挂上警告标志
5. 为压缩机完全降压
6. 关闭空压机通向供气系统的截止阀

> ⛔ 禁止在机器运行或带压状态下拆卸螺母、加油塞或其他零件。
> ⛔ 禁止使用汽油、煤油等可燃溶剂清洗零部件。
> ⛔ 禁止使用明火检查机器或压力容器内部。
> ⛔ 禁止在润滑油系统附近进行焊接或其他发热操作。

### 5. 操作指南页（Operating Procedures）

**首次安装启动流程（共10步）：​**

1. 确认进气/环境温度 ≥ 0°C
2. 拆除主机、电机及油分桶的红色运输固定螺栓（若需重新运输须重新安装）
3. 检查电气连接是否规范，电缆连接端子是否牢固可靠
4. 检查气路连接是否可靠
5. 调试前确保无人滞留在电机或压缩机旁的危险区域
6. 运行时必须关闭所有门和面板
7. 首次接电须检查驱动电机转向，转向错误立即按紧急停机按钮（否则可能严重损坏压缩机）
8. 接通总电源后，所有指示灯和液晶显示器将点亮进行显示自检
9. 机器配有自动运行监测系统，故障时自动停机
10. 按下启动按钮开始运行

### 7. 故障排除页（Troubleshooting）

**建议以表格形式完整呈现8类故障：​**

| 故障现象 | 可能原因 | 解决方法 |
|---|---|---|
| **无法启动** | 保险丝烧毁 / 保护继电器动作 / 按钮接触不良 / 电压过低 / 电机或机体故障 | 电气人员检修更换；机体故障时手动盘动，无法转动则联系厂家 |
| **排气温度高（超105°C停机）​** | 环境温度过高 / 温控阀失灵 / 润滑油不足 / 冷却器翅片过脏 / 油过滤器阻塞 / 冷却风扇故障 / 温度传感器故障 | 改善通风 / 检查更换温控阀 / 调整油位 / 清洁冷却器翅片 / 更换油过滤器或冷却风扇 |
| **排气压力低** | 空气需求超过供给 / 空气过滤器阻塞 / 进气阀未完全打开 / 油气分离芯阻塞 | 检查管道泄漏 / 清洁或更换滤芯 / 检查进气阀 / 必要时更换油气分离芯 |
| **排气压力超压停机** | 压力变送器失灵或异常干扰 / 出气阀门关闭 | 排除或更换变送器 / 打开出气阀门 |
| **无法空车 / 空车时压力继续上升** | 进气阀动作不良 / 压力传感器失效 / 泄放阀失效 | 检查进气阀 / 检修或更换传感器及泄放阀 |
| **空重车频繁** | 管路泄漏 / 空重车压差设定过小 / 空气消耗量不稳定 | 检查管路 / 重新设置压差 / 增加储气罐容量 |
| **空气含油量高 / 油耗过量** | 油位过高 / 回油管堵塞 / 油气分离芯或垫圈破损 / 润滑油系统泄漏 / 排气压力过低 / 润滑油泡沫过多 | 检查油位 / 清洗回油管 / 检查更换分离芯 / 提高排气压力 / 更换润滑油 |
| **停机时油雾从空滤冒出** | 进气阀不严 / 重车停机 / 最小压力阀泄漏 / 油细分离器破损 | 检查进气阀 / 检修最小压力阀 / 更换油细分离器 |

### 8. 维护保养页（Maintenance & Care）

#### 日常 / 定期检查计划

| 周期 | 检查内容 |
|---|---|
| **每日** | 检查油位及显示屏读数；水冷机组检查冷却水量 |
| **每周** | 检查异常声响和泄漏；核对仪表读数；检查温度显示；清洁空气滤芯（用 0.2～0.4 MPa 压缩空气从内向外吹） |
| **每月** | 检查锈蚀与松动，去锈上油或涂漆；排放冷凝水 |
| **每三个月** | 检查冷却器并清洁；检查空气过滤器滤芯；检查软管老化情况；检查电器元件，清洁电控箱 |

#### 关键部件更换周期表

| 部件 | 更换 / 检查周期 | 备注 |
|---|---|---|
| 空气过滤器滤芯 | 清洁：每周；更换：1,000～1,500 小时 | 恶劣环境（矿山/陶瓷厂/棉纺厂）每 500 小时更换 |
| 机油过滤器 | 首次：500 小时；第二次：1,000 小时；之后每 2,500 小时 | 每次换油时必须同时更换 |
| 压缩机润滑油 | 首次：500 小时；之后每 2,000～2,500 小时或每年 | 严禁不同品牌混合使用 |
| 油气分离器 | 约 3,000 小时，或前后压差超过 0.12 MPa 时立即更换 | 超期使用将导致电机过载 |
| 进气阀活塞密封圈 | 每 2,500 小时检查或更换 | — |
| 最小压力阀 | 每 2,500 小时检查开启压力并清洗 | — |
| 冷却器除尘 | 每班次用 ≥0.4 MPa 干燥压缩空气吹扫 | 油污环境须使用结碳清洗剂 |
| 安全阀 | 每 4,000 小时检查动作灵敏性 | — |
| 电机轴承注脂 | 按电机使用说明执行 | 部分机型需要定期注油 |
| 主机大修 | 每 10,000 小时 | 必要时更换径向轴承和止推轴承 |

#### 润滑油使用规范（重要警示）

- 严禁混合使用不同品牌或类型的润滑油
- 严禁使用过期润滑油（会导致润滑性下降、闪点降低，极易引发高温停机及油品自燃）
- 强烈建议使用原厂专用螺杆机油
- 冷凝水须定期排放：一般每周一次，高温气候下每 2～3 天一次
- 停机 4 小时以上须在无压状态下打开放油阀排放冷凝水，见到机油流出时立即关闭阀门

#### 换油操作步骤

1. 启动空压机运行 5 分钟，使油温升至 50°C 以上
2. 停机，在油气桶存有 0.1 MPa 压力时，缓慢打开底部放油阀，接上储油罐
3. 待润滑油呈滴状后关闭放油阀，拧开油滤芯放尽管路余油，换上新油滤芯
4. 打开加油口，注入新油至油标刻度线范围内，拧紧后检查是否有渗漏

#### 空气滤芯清洁步骤

1. 拧开压盖锁扣，取出空滤芯
2. 用 0.2～0.4 MPa 的压缩空气从空滤芯内腔向外吹除外表面尘埃颗粒
3. 用干净抹布擦净空滤壳内壁脏物
4. 回装空滤芯，注意前端密封圈须与空滤壳内端面贴合严密

#### 油气分离器更换步骤

1. 拆下油气桶盖上安装的各控制管接头，取出回油管，拆出油气桶上盖紧固螺栓
2. 移开油气桶上盖，取出油细分离器，除去粘在上盖板上的石棉垫及污物
3. 装入新的油细分离器，注意安装必须同心，确保底部密封环完全接触，锁紧顶部螺丝时不要让油分芯变形
4. 按原样装回上盖板、回油管、各控制管，检查有无泄漏

#### 周期性保养计划（按运行小时）

- **每 100 小时（或一周内）：​** 清洗油过滤器一级滤网和油箱粗滤网
- **每 300 小时：​** 清洗空压机过滤器芯子，清洗后吹干并浸泡同型号机油；清洗或检查减荷阀、最低压力阀、自动放空阀
- **每 1,000 小时：​** 更换油过滤纸质滤芯；检查安全阀、温度控制阀、压力调节器、压力调节阀工作状态
- **每 3,000 小时：​** 根据油的乳化程度、杂质多少、粘度变化酌情换油；清洁油冷却器（一般要求每 4 个月清洗一次）
- **每 10,000 小时：​** 对主机进行大修，必要时更换径向轴承和止推轴承
